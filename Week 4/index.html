In the third and fourth line of the style box, we created 3 rows and 3 columns of 5 rem of side to create the board, and we set it at the center of the screen with justify-content: center.

We deleted the board perimeter borders using:

.boardItem:nth-child(3n) {

border-right: 0;

}

.boardItem:nth-child(1n) {

border-left: 0;

border-top: 0;

}

.boardItem:nth-child(n+7) {

border-bottom: 0;

}

Using (3n) in the first and second line, we delete the right border of each box multiple of 3

Using (1n) in the fourth to sixth line, we deleted the left and top borders of the all boxes

Using (n+7) in the eighth and ninth line, we deleted all the bottom borders of each box after the seventh box element.
